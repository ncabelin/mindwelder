How it Works

Welcome to the Mindwelder Blogs. We make it easy to create fill-in-the-blank Q & A's in your blog posts. All you have to do is underline style an area of your post and you got an answer.

Get Started :

  • 1. Add a post in either Rich Text mode (What You See Is What You Get or WYSIWYG editor), or edit in HTML mode if you are comfortable with HTML markup syntax (HTML mode gives you a lot more options but you can switch between the two editors)
  • 2. You can format the post as bold / italic (b and i tag), font-size (h1,h2,h3), insert code (pre,code), add a link(a tag) or image(img tag) on the toolbar at rich text mode
  • 3. To define a section as an answer in Rich Text mode, select the text and click the U button to underline it.
  • 4. Click to review / read the post, to view it in TEST mode.
  • 5. When in TEST mode the answers will appear as blank inputs initially, if you click on the button while the input field is blank the answer will be displayed, giving you a chance to review the answer, otherwise whatever you typed will be checked.
  • 6. If what you typed is the same as the answer(not case-sensitive) it will be marked as correct and added to the total score, otherwise its font color will turn red (then you can try again, clear the input field and click on the button to review).



We hope you find the application educational and helpful. Have fun learning!
